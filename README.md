# HealthCare-Data-Analysis-Excel
Healthcare Data Analysis project using Excel including data cleaning,feature engineering ,and exploratory data analysis on 50k+ patient records.

📌 Project Overview
This project analyzes a healthcare dataset containing patient records, billing details, medical conditions, and hospital information.

🛠 Tools Used
-Microsoft Excel
-Pivot Tables
-Data Cleaning Functions (TRIM, UPPER, IF, DATEDIF)

📊 Data Cleaning Steps:
-Removed duplicates
-Standardized text values
-Formatted date columns
-Fixed billing decimal precision
Created derived columns:
-Length_of_Stay
-Age_Group
-Billing_Category
-Admission_Month
