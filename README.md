# HealthCare-Data-Analysis-Excel
Healthcare Data Analysis project using Excel including data cleaning,feature engineering ,and exploratory data analysis on 50k+ patient records.


🏥 Healthcare Data Analysis Dashboard (Excel)
📌 Project Overview
This project is an industry-level Healthcare Data Analysis Dashboard built using Microsoft Excel.
The objective was to transform raw healthcare data into meaningful business insights using structured data cleaning, pivot analysis, KPIs, and interactive dashboard design.

🔄 Project Workflow (How the Work Was Done)

1️⃣ Data Cleaning
Removed duplicate records
Standardized date formats
Handled missing values
Created helper columns (Month_Year, Length_of_Stay)
Ensured correct data types (Date, Currency, Text)

2️⃣ Data Transformation
Created calculated columns:
Length of Stay = Discharge Date – Admission Date
Extracted Month & Year from Admission Date
Structured dataset for pivot-based analysis

3️⃣ Pivot Table Creation
Created multiple pivot tables for:
Monthly Revenue Analysis
Total Patients per Month
Doctor-wise Revenue
Insurance-wise Coverage
Admission Type Distribution
All pivots were connected to slicers for dynamic filtering.

4️⃣ KPI Development
Built key performance indicators such as:
Total Revenue
Total Patients
Average Revenue per Patient
Average Length of Stay
Top Performing Doctor
Top Insurance Provider
KPIs dynamically update using pivot-based calculations.

5️⃣ Dashboard Design
Combined KPIs and charts into a single Dashboard sheet
Implemented slicers (Month, Doctor, Insurance, Admission Type)
Used clean formatting, aligned visuals, and minimal color palette
Designed for executive-level reporting
